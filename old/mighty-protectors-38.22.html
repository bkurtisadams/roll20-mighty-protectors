Attack Notes Shorthand
SAVE ATTACKS
sav:BC:mod:rec
• BC = Save type (EN, AG, IN, CL)
• mod = Save modifier (+5, -3)
• rec = Recovery modifier (optional)
Example: sav:EN:+5:-14
SNARE ATTACKS
snr:type:bp/max
• type = grp (grapnel) or ice
• bp = Break Point
• max = Max BP (optional)
Example: snr:ice:9/15
ARMOR PIERCING
ap or ap:N
• ap:N = Reduces target's protection by N
• ap = Ignores ALL protection & Invuln
• Hardened negates AP 1:1
• Leftover AP (exceeding prot) bypasses Invuln
Example: ap:5 vs 8 prot = 3 effective prot
PUSHING (2 PR)
• Declare before rolling
• Standard Attack: +2 damage
• Save Attack: -2 to target number
• Snare Attack: +2 to BP
• Movement: x2 Accel & Top Speed
• Strength: +2 HTH, x2 Carry
Cannot push Gear-based abilities
DEFENDER NOTATION
h suffix: Hardened (e.g. 5h). Negates AP 1:1.

Combat Query Reference
RANGE (4.7.3)
0-4": 0 | 5-9": -1 | 10-19": -2 | 20-39": -3 | 40-79": -4 | 80-159": -5 | etc. (doubles each bracket, -1 each)
CALLED SHOTS (4.14.2)
• Head -6: Double hits taken after protection
• Arm/Leg -3: EN+7 save or lose use; AG save or drop/fall
• Avoid Light Armor -3 | Heavy Armor -6
• Gear -3: Damage vs breakpoint
DEFENSE (4.7.1)
Subtract target's Physical Def (AG-based) or Mental Def (IN-based) from to-hit.
STANCE (4.14.3)
• Defensive: +3 Def, -3 to hit
• Full Defense: +6 Def, ½ Move, costs action to adopt
AIM (4.14.1)
Ranged only, must be prone/braced. 1 move phase: +2 | 3 phases: +4 | 7 phases: +6. Lose bonus if target moves out of sight.
PUSH (5.5)
Cost: 2 PR. Declare before rolling. Not for Gear.
• Standard: +2 damage | Save: -2 target number
• Snare: +2 BP | Move: x2 Accel & Speed | ST: +2 HTH, x2 Carry
MULTI-ACTION (4.5.1)
-3 per extra action. 2 actions: -3 each | 3 actions: -6 each | etc.
